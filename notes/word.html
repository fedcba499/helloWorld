Shortcuts in word

ctrl + E -> indenting line to center

ctrl + L -> indenting line to Left

ctrl + R -> indenting line to Right

ctrl + B -> Bolds the line

ctrl + U -> Underlines the line

ctrl + I -> Italises the line

ctrl + F -> Find in the document

ctrl + H -> Replace in the document

ctrl + shift + left/ right arrow -> Extend Current Selection
